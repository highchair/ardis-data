Sr. Software and Systems Design Engineer

Position Description:

Design software systems. Translate business requirements into technical solutions. Assist in the ongoing assessment of existing and proposed business systems. Develop strategies to evolve systems. Collaborate with internal and external business partners and application engineers to translate strategies into technical solutions. Work with employees and customers to extract insights and measure progress to inform future development. Develop and maintain systems documentation, diagrams, changes, and standard operating procedures. Perform software engineering design and build client-side and server-side architecture. Define, design, and develop RESTful API endpoints by integrating with a variety of third-party APIs and data sources to achieve core business goals.

Implement advanced JavaScript-driven interfaces, integrate with APIs, build beautiful data visualizations. Design and implement client-facing mobile application. Develop testing methodologies and execute testing actions on code and processes. Assist in code deployments, fixes, updates, and related processes. Collaborate and keep track of code changes using Git for version control. Perform server and database administration. Develop internal procedures and technical solutions for communications, calendaring, workflow management, compliance, and information security.

Position Duties:

Employer will accept a masters degree in computer science and two (2) years of experience in a software engineer related occupation. Position requires two (2) years of experience in each of the following:

  1. Microsoft SQL server, mysql, or PostGreSQL database management systems performing data analysis, and writing complex SQL queries
  2. Backend development using at least one of the following: Python, PHP, C#, or Node.js
  3. Javascript, HTML, CSS and related tools and packages
  4. Frontend web development using Javascript, Angular, Vue, or React
  5. Native or hybrid mobile application development
  6. Version control system (GIT)
