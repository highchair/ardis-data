Sr. Software and Data Engineer

Position Description:

Build extraction frameworks to collect data from various external sources using Python and shell scripting. Build a framework to scrape data from sources using Selenium, BeautifulSoup and Python. Build framework to automatically test data quality using Python. Build framework to ingest data into MS SQL Server, PostgresSQl etc. using Python. Automate code generation process using Python. Implement Git integrated CI/CD solutions to automatically build test and deploy code across various environments (DEV, QA, UAT, PRD). Build a code generation framework to automatically create tables and table definitions in databases using SQL scripts and Python.

Perform data analysis. Analyze and profile raw data in data lake, measure data quality based on accuracy, relevancy and completeness, orderliness, timeliness, and uniqueness using Python, Pandas, SQL, etc. Create dashboards to monitor data quality of sources using Python, SQL, and Power Bi. Build advanced data profiling frameworks using Python and SQL to discover important aspects like structure, content and relationship. Create a framework to analyze data flows and determine source to target data lineage and store data into a metadata library using automation. Perform data sourcing. Develop robust software tooling to manage data. Build API endpoints using Python, Flask, Node, etc.

Position Duties:

Employer will accept a Masters Degree in data analytics or data science and two (2) years of experience in a data engineer or data analyst related occupation. Position requires two (2) years of experience in each of the following:

  1. Architecting extract, transform, load jobs
  2. Building cloud-based scalable solutions
  3. Implementation of machine learning models including Scikit-learn, Auto-ML, and statistical modeling
  4. Data profiling, descriptive statistics, data visualizations, data wrangling ,and advanced analytics
  5. Automation tools and frameworks including Selenium
  6. Building real time applications using streaming frameworks including Kafka
  7. Building microservices using Flask and Python
  8. Python
  9. Microsoft SQL server, MySQL, or PostGresSQL-database management systems
  10. Building data pipelines and scheduling jobs utilizing Python, SSIS, Airflow, and shell scripting
